# **Статистический анализ данных: сервис аренды самокатов**

**Цель проекта:**
- проанализировать данные аренды самокатов.
- проверить гипотезы, направленные на улучшение и рост эффективности сервиса.

**Описание проекта:**

Загрузили данные сервиса о пользователях, их поездках и подписках. 
Провели EDA. 

Проверили гипотезы:
- тратят ли пользователи с подпиской больше времени на поездки.
- расстояние одной поездки в 3130 метров.
- помесячная выручка от пользователей с подпиской и без равны.

Для проверки гипотез применяли t-критерий (для двух незасивисмых выборок и одновыборочный).

**Ключевые слова:**

EDA, Тестирование гипотез, ttest_ind, ttest_samp
